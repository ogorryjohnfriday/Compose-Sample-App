Compose Sample App

A simple Android application built with Kotlin and Jetpack Compose, designed to demonstrate UI design, state management, and clean architecture basics.

This project serves as a lightweight example of how I build Compose UIs, manage state, and structure Android apps using modern development tools.

Features

Jetpack Compose UI

State management

ViewModel (MVVM pattern)

Material 3 components

Simple navigation (optional)

Clean project structure

 Tech Stack

Kotlin

Jetpack Compose

Android Studio

ViewModel (Jetpack)

Coroutines (optional)

Material Design 3
